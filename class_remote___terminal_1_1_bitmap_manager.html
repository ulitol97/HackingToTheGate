Detailed Description

The BitmapManager class is in charge of handling the conversion from the Bitmap objects received from the VNC server to Unity suitable sprites or textures that can be rendered in-game by the engine.

The usage of the System.Drawing namespace imposed by the VNC API limits the platforms in which the game may run to Windows operating systems
