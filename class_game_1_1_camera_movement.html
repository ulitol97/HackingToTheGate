Detailed Description

The CameraMovement class extends the functionality of a Unity camera component by making the camera locate a target to follow and move towards that target if needed.

void Game.CameraMovement.LateUpdate ( )
private

Function called on each frame the CameraMovement script is present into the game. It is is charge of arranging the camera position updates./>

LateUpdate was used instead of Update to give time to all game objects (including the camera target) to update their positions before the camera follows them.
